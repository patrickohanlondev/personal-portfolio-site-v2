I created this project as a learning exercise in vanilla JavaScript.

I took advantage of the many resources outlining the fundamental design of a todo list app. Instead of creating a simple todo list I wanted this app to have a more focused goal, keeping track of films and TV show the user has watched.

As my skill in JavaScript develops I plan to create a more fully featured version of this app complete with API integration, allowing for film artwork, release years and genres to be auto-populated.
